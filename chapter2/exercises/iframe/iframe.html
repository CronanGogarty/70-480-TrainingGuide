Some examples of working with the <iframe> tag.

An iframe embedding content from an external resource www.theguardian.com/uk;

An iframe embedding local content from internal-iframe-content.html;

An iframe embedding local content from internal-iframe-content.html this time with the sandbox="sandbox" attribute;
In this instance the scripts within the embedded pages do not run.

An iframe embedding local content from internal-iframe-content.html this time with the sandbox="allow-scripts" attribute;
In this instance the scripts within the embedded pages will run with the exception of the pop-ups.

An iframe with the seamless="seamless" attribute. Note that support for this attribute is limited.
Of course it is possible to style the iframe using CSS as I have done here by changing the default border to red.
In addition, I have enabled allow-same-origin allow-form allow-scripts in the sandbox attribute to increase functionality.
